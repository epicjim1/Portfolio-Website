My Projects

This is where some of my major and most impactful projects are displayed! Most of my projects focus primarily on Game Development including frameworks and tools such as Unity, Visual Studio Code, Blender and so much more! My projects also showcase the Model-View-Controller framework, Google Ads, In-App Purchasing, and most importantly, fun user interaction! I also often dabble with firmware and embedded systems.

To get a full list of my projects, please visit my GitHub!

01
Ball Drop Mobile

This is a mobile version of the BallDrop game I made during the panademic to learn how to program. It is a 2D physics based game where the player must get their ball into a black hole by moving various platforms and avoiding obstacles, similar to cut the rope. I used the Unity game engine with C# to create the foundation and used later used Googles Ads API, Swfit, and XCode to integrate it over to the IOS framework. All models, effects, and sounds were handcrafted create the most immersive experience I could give. This game has been played by 600+ users and was on the top 50 Puzzle games list on the AppStore. 03
Tic-Tac-Toe MVC Game

A childhood favourite! I recreated the classic Tic-Tac-Toe Game using the Java Swing GUI and through implementing the Model-View-Controller design pattern. In addition, I added my own twists on the game by implementing a difficulty feature that impacts AI intellegence, a round feature that sees a player play a certain number of rounds before a game is over, and a dynamic file saving and viewing system to see past game results. This game was played by 50+ high school students and was used in Computer Science classroom demonstrations. Click on this link to access the project files!

Unreleased machine learning sim

This project was about learning to implement machine learning on 3D objects.
